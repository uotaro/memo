# アプリアイコンの作り方

Figma初めてでも迷わないように「Flutter用アイコン（iOS＋Androidアダプティブ）」を**完成まで**の手順で書きます。  
（やることは *1024角のキャンバスを作る → 背景作る → ロゴ置く → 影 → 書き出し（背景/前景/iOS合成）* だけです） 
  
※ chatGPT による回答

---

## 0. Figmaを開いたら最初にやること（新規ファイル）
1. Figmaを開く
2. 左上の **New design file**（新規デザイン）をクリック  
3. 画面中央が白いキャンバスになったらOK

---

## 1. 1024×1024 の「台紙（Frame）」を作る
1. キーボードで **F**（Frameツール）
2. 右側パネルの **W / H** にそれぞれ  
   - **W: 1024**
   - **H: 1024**
3. キャンバス上をクリックしてフレームを置く  
4. 左側のレイヤ一覧に `Frame 1` ができればOK  
   - 分かりやすく、`icon_1024` にリネームしておくと良い  
     - レイヤ名をダブルクリック → `icon_1024`

---

## 2. 背景（Background）を作る（不透明）
1. キーボードで **R**（Rectangle）
2. さっきの `icon_1024` フレームの中をドラッグして四角を作る  
   - きっちり全面にしたいときは、四角を選択して右側で  
     - **X: 0 / Y: 0 / W: 1024 / H: 1024**
3. 右側パネルの **Fill（塗り）** をクリックして色を選ぶ  
4. レイヤ名を `background` にリネーム

> コツ：背景はまず「単色」がおすすめ。ロゴ中心アイコンは単色が一番綺麗に見えやすいです。

---

## 3. ロゴを置く（Foregroundの中身）
ロゴの元データが何かで操作が変わります。どれでもOKです。

### A) SVGがある場合（最強・おすすめ）
1. SVGファイルを **ドラッグ＆ドロップ**でFigmaに入れる  
2. 自動で配置されるので、それをフレーム内に移動 
  
※ SVGファイルは [Bootstrap icon サイト](https://icons.getbootstrap.com/) でも入手可能


### B) PNG（透過）しかない場合
1. PNGを **ドラッグ＆ドロップ**
2. フレーム内に移動

### C) まだロゴがない場合（仮で文字ロゴを作る）
1. キーボードで **T**（Text）
2. クリックして `A` やアプリの頭文字を入力
3. 右側でフォントサイズを大きく（例：300〜450）

---

## 4. ロゴを中央に揃える（ここ大事）
1. ロゴ（文字 or 画像）をクリックして選択
2. **Shift** を押しながら `icon_1024` フレームもクリック（2つ選択状態）
3. 上部ツールバーの整列で
   - **Align horizontal centers（横中央）**
   - **Align vertical centers（縦中央）**

---

## 5. ロゴのサイズを“安全に”する（切れ対策）
1. ロゴを選択
2. 右側の **W / H** を見ながら調整（角のハンドルをドラッグでOK）
3. 目安：ロゴ本体がだいたい **560〜640px** くらいに収まるように  
   - まずは **600px前後** を狙うと失敗しにくいです

---

## 6. 影をつける（Foregroundに影＋透過）
1. ロゴを選択
2. 右側パネルで **Effects** を探す
3. **+** を押して **Drop shadow** を追加
4. 影を2段にしたいので、もう一回 **+** を押してDrop shadowをもう1個追加

おすすめ設定（そのまま入力でOK）：
- 影1：X 0 / Y 12 / Blur 24 / Opacity 18%
- 影2：X 0 / Y 4 / Blur 8 / Opacity 12%

> 影の色は黒のままでOK。強すぎるならOpacityを下げれば綺麗になります。

---

## 7. 書き出し用に「Foregroundだけ」「Backgroundだけ」「iOS合成」を作る
ここが一番混乱しがちなので、手順どおりにやればOKです。

### 7-1) Foreground用（透過）を書き出す準備
- Foregroundは「ロゴ＋影だけ」です。背景は含めません。

手順：
1. `background` レイヤの **目のアイコン（表示）** をOFF（非表示にする）
2. フレームの中が透明の市松模様っぽく見えればOK（見えなくても気にしないでOK）
3. ロゴが見えている状態にする

### 7-2) Background用（不透明）を書き出す準備
- Backgroundは「背景だけ」です。ロゴは含めません。

手順：
1. `background` を表示ONに戻す
2. ロゴレイヤを **目アイコンOFF** で非表示にする

### 7-3) iOS用（不透明合成）を書き出す準備
- iOSは「背景＋ロゴ＋影」を**合成して不透明**が安全です。

手順：
1. `background` 表示ON
2. ロゴ表示ON

---

## 8. 書き出し（Export）のやり方（Figma操作）
Figmaの書き出しは「右下のExport」だけ覚えればOKです。

### Foreground（透過PNG）を書き出す
1. ロゴ表示ON、背景表示OFFの状態にする（7-1の状態）
2. 右側パネル一番下の **Export** を開く
3. `+` を押す（Export settingsを追加）
4. Formatを **PNG** にする
5. **Export** ボタンを押す
6. ファイル名：`foreground.png`

### Background（不透明PNG）を書き出す
1. 背景表示ON、ロゴ表示OFFの状態にする（7-2の状態）
2. 同じく Export → PNG → Export
3. ファイル名：`background.png`

### iOS用（不透明合成PNG）を書き出す
1. 背景表示ON、ロゴ表示ON（7-3の状態）
2. Export → PNG → Export
3. ファイル名：`ios.png`

> 書き出し対象は基本「フレーム（icon_1024）」を選択してExportすると分かりやすいです。  
> 左のレイヤで `icon_1024` をクリックしてからExportしてください。

---

## 9. Flutterへ入れる配置（おすすめ）
Flutterプロジェクトにこんな感じで置くと管理が楽です。

```
assets/icon/background.png
assets/icon/foreground.png
assets/icon/ios.png
```

---

## 10. よくあるつまずき（ここだけ先に潰す）
- **Exportが見当たらない**  
  → 右側パネルを下までスクロール。いちばん下にあります。
- **背景を消したのに透明にならない**  
  → 背景レイヤを非表示にしても、フレーム自体にFillが付いていると透明になりません。  
  対処：`icon_1024` フレームを選択 → 右側のFillがあればOFF。
- **ロゴがズレる**  
  → フレームとロゴを同時選択して、整列ボタンで中央揃え。
